## System Description: Institute Management System (IMS)

The IMS is designed to automate management processes of an educational institute — from student admission to examination management, course scheduling, human resources, and salary management.

**Goal:**  
- Model the **entire system conceptually**
- Implement:
  - Dashboard (mandatory)
  - End-to-end features of **two specific modules**

---

## Modules of IMS

| Time Table        | Messaging        | Attendance       |
|------------------|-----------------|-----------------|
| Student Admission| Courses & Batches| Examination      |
| Human Resources  | User Management | News Management  |
| Student Details  | Finance         | Multiple Dashboards |

---

## Detailed Capabilities

### Dashboard
- Search-based navigation
- User-friendly interface
- Displays latest news & login info
- Configurations (language, currency, time zone)
- Manage courses, batches, subjects
- SMS module integration
- Manage student categories

---

### Admission
- Unique student ID generation
- Multiple guardians support
- Upload student photos
- Customizable forms

---

### Student Details
- Batch-based student view
- Search for current and former students
- Advanced filtering

---

### Examinations
- Create exams (grades/marks/custom)
- Automated report generation
- Graphical analytics
- GPA, CCE, CWA support

---

### Manage Users
- Global search
- View/edit user profiles
- Role-based access control

---

### Human Resources
- Employee lifecycle management
- Payroll system
- Leave management
- Employee search

---

### Attendance
- Easy marking with remarks
- Multiple report formats
- Filterable reports

---

### Finance
- Fee classification & collection
- Defaulter analysis
- Expense/income tracking
- Automated transactions
- Tutor assignment for tracking

---

### Messages
- Messaging system for all users
- Record communications
- Broadcast announcements

---

### Time Table
- Drag-and-drop scheduler
- Real-time alerts
- Edit/delete timetables

---

### Manage News
- Create/edit news (RTF)
- Comment system
- Moderation tools
